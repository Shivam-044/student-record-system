A robust, command-line interface (CLI) application built with Python to efficiently manage student records. This project demonstrates core programming concepts including **Object-Oriented Programming (OOP)**, **File I/O operations**, and **Exception Handling**.

---

## 🌟 Key Features

- **Persistent Data Storage**: All records are saved locally in a structured text format.
- **Input Validation**: Prevents crashes by validating numerical inputs and handling missing files.
- **Clean UI**: A user-friendly menu system for seamless navigation.
- **Scalable Architecture**: Built using classes to allow for easy future integration with SQL databases.

## 🛠️ Technical Stack

- **Language:** Python 3.x
- **Storage:** Flat-file database (`data.txt`)
- **Concepts:** OOP, Error Handling, String Formatting
